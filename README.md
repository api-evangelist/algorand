# Algorand

Algorand is a pure proof-of-stake blockchain platform designed for speed, security, and decentralization. It exposes REST APIs through two core daemons — Algod (the Algorand protocol daemon) for real-time node interaction including submitting transactions, querying account balances, retrieving blocks, and managing smart contracts; and the Indexer daemon for deep historical querying of accounts, assets, applications, transactions, and blocks from a PostgreSQL-backed store. A third daemon, KMD (Key Management Daemon), handles local key storage and signing.

**Human URL:** https://dev.algorand.co/

**Base URLs:**
- Algod (mainnet): https://mainnet-api.4160.nodely.dev
- Indexer (mainnet): https://mainnet-idx.4160.nodely.dev

## APIs

- **Algorand Algod REST API** — Real-time node interaction: accounts, assets, blocks, transactions, applications, TEAL compilation, state proofs
- **Algorand Indexer REST API** — Historical ledger analytics: search transactions, accounts, assets, applications, blocks
- **Algorand KMD REST API** — Local key management and transaction signing daemon
